## インストール方法
Ipe 公式サイトの **Download the current Ipe version** から OS にあったものをダウンロードしてください．

公式サイト: https://ipe.otfried.org/
### 権限の設定 (macOS の場合)
macOS の場合，ダウンロードしたアプリを開こうとしたときにエラーが出る場合があります．
そのような場合，`システム設定 > プライバシーとセキュリティ` から実行を許可してください．

## tex の設定

### macOS の場合
#### latex の環境構築
mactex をインストールします．
https://texwiki.texjp.org/?MacTeX

#### tex の紐づけ
ホームディレクトリの `~/.ipe` ディレクトリに，`ipe.conf` ファイルを設置します．
内容は，
```
IPELATEXPATH=/Library/TeX/texbin
```
としてください．

↓ 以下のコマンドを実行すればできます．
```shell
$ echo 'IPELATEXPATH=/Library/TeX/texbin' > ~/.ipe/ipe.conf
```

### luatex の場合
#### engine の設定
Ipe を開き，`Edit > Document Properties` から Latex engine を **luatex** に変更します．

![環境構築_document_properties.png](/img/user/images/%E7%92%B0%E5%A2%83%E6%A7%8B%E7%AF%89_document_properties.png)

#### preamble の設定
Latex preamble を以下のように設定します．

```tex
\usepackage{fontspec}
\setmainfont{Harano Aji Gothic}
```

以上で日本語設定は完了です．
